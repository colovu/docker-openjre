# 简介

针对 OpenJRE 的 Docker 镜像，用于提供基础 Java 运行环境。

使用说明可参考：官方说明



**JRE 版本信息：**

- 8u262

**镜像信息：**

- 镜像地址：registry.cn-shenzhen.aliyuncs.com/colovu/openjre:latest

  

## **TL;DR**

基本验证命令：

```shell
# 11之前的版本
$ docker run -it --rm registry.cn-shenzhen.aliyuncs.com/colovu/openjre java -version

# 11之后的版本
$ docker run -it --rm registry.cn-shenzhen.aliyuncs.com/colovu/openjre java --version
```
